### 系統規則:
採用選單式操作，使用者需透過輸入數字選擇功能進行操作。系統提供航班查詢、訂票、取消訂票以及訂單查詢等功能。
在查詢航班時，使用者可選擇查看所有航班，或依航空公司篩選特定航班資訊。進行訂票時，需輸入航空公司以及航班號碼，系統會檢查該航班是否存在並進行訂票；若訂票成功，則會建立一筆訂單紀錄。
取消訂票時，使用者需輸入相同的訂票資訊，系統會檢查是否存在對應的訂單，若存在則取消該訂單並更新相關資料。使用者亦可透過查詢功能查看目前所有已訂機票的紀錄。
### 程式介紹
系統主要分為三個類別，分別為 Flight、Booking 以及 BookingSystem。Flight 類別用來儲存航班的基本資訊，包含航空公司、航班號、出發地、目的地與座位數；Booking 類別則負責記錄乘客的訂票資訊，包含航空公司與航班號，用於保存每一筆訂票紀錄；BookingSystem 類別，負責管理所有航班與訂單資料，並提供航班查詢、訂票、取消訂票以及訂單查詢等功能。
